- [Node.js](https://nodejs.org/) >= 18  
- [Expo CLI](https://docs.expo.dev/get-started/installation/)  
- [Expo Go App](https://expo.dev/client) (on your phone, Android/iOS)  

Install Expo CLI globally:  
```bash

```
